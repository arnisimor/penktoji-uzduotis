### Penktoji-uzduotis
## zodziu skaiciuokle
- sukurti tekstinį failą su numatytu pavadinimu (duomenys).
- įterpti tekstą į anksčiau minėtą failą.
- paleisti programą.
- rezultatai bus išvesti į "url.txt" failą ir "rezultatai.txt".
